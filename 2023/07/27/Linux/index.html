知识点

绝对路径：从**/**开头的路径 /home/user1

相对路径：相对于当前目录开始

linux一些命令

ctrl快捷键

ctrl + D 关闭当前进程

命令

stat

stat + 文件名 用于查看文件的信息

pwd

pwd查看当前所在目录

alias 命名

自己设置命令名称：

alias + 名称=”” （输入命令） 例如:

alias wd="vim /etc/sysconfig/network-scripts/ifcfg-ens160"

取消： unalias +设定的名称 例如：

unalias wd

tree

tree＋文件夹，可以展示出目录的结构

touch –创建

touch + 文件名 创建空文件

如果文件存在，则修改这个文件的时间

vim –创建

vim + 文件名 创建空文件

重定向： echo aaa > xuegod.txt 创建一个名为xuegod.txt的文件，并且内容为 aaa

mkdir –创建

mkdir + 名称 创建文件夹

在创建一个目录的时候，如果这个目录的上一 级不存在的话，要加参数-p

即： mkdir + -p + 名称

rm –删除

rm + 文件名称 删除某个文件

-f 强制删除，没有提示

-r 删除目录，递归删除，把目录下的文件和文件夹也删除

注意 ： rm + 目录名称 这个操作无法执行，需要加上 -r

如 rm -r aaa ， 该命令一般和rf一起用： rm -rf +名称

cp –复制文件

cp 源文件/目录 目标文件/目录 即 ： cp 想让文件复制到的地点 复制的文件

选项：-R/r：递归处理，将指定目录下的所有文件与子目录一并处理。复制目录时会使用 r 参数 例子：

cp -r /boot/grub2 /opt/ #复制目录（文件夹） 意思： 将opt这个文件夹的所有内容包括opt本身，复制到该位置/boot/grub2

选项：\ ：覆盖文件，不提示 放在cp前，即：\cp

-p：复制文件会保留文件属性 复制文件的时候，会包括其权限等

\cp -r /opt/!(dir1|dir2) /root/ #除了 dir1 或 dir2 以外其他文件都 复制到 root 目录下

mv –移动文件

mv + 想要移动的文件路径 + 移动的目的地路径

mv passwd bbbpasswd #重命名 即 将文件名称passwd 改为bbbpasswd

mv bbbpasswd /opt/passwd 将名为 “bbbpasswd” 的文件移动到 /opt/passwd 目录下，并将文件名改为 “passwd”

cat、more、less –文本查看命令

cat、more、less + 文件名称

不同点：cat：会直接将文件的内容输入到终端

more：可以一页一页查看文件的内容，但是不支持后退，但几乎不需要加参数，空格键是向下翻页，Enter 键是向下翻一行

less：支持前后翻滚，既可以向上翻页（pageup 按键），也可以向下翻页（pagedown 按键）。，空 格键是向下翻页，Enter 键是向下翻一行

head –文本查看命令

head、tail + 文件名 用于查看文件的部分内容

head：查看文件的前十行

参数: -n 显示从文件头开始的行数

head -n 3 /etc/passwd #显示前 3 行

tail –文本查看命令

语法:tail (选项) 文件名

tail：查看文件的后十行内容

参数: -n 显示文件尾部多少行的内容(n 为数字)

-f 动态显示数据（不关闭）,常用来查看日志

Vim

在vim编辑的正常模式中

技巧

按u键撤销刚才的操作

ctrl+r 恢复撤销之前的内容

进入编辑模式：说明：

i 当前字符之前插入 (光标前)

I 行首插入 (行首)

a 当前字符之后插入 (光标后)

A 行尾插入(行尾)

o 下一行插入 (另起一行)

yy 复制整行

复制 N 行： Nyy ，比如： 2yy ，表示复制 2 行

dd（删除，以行为单位，删除当前光标所在行） 删除 N 行： Ndd ，比如： 2dd ，表示删除 2 行

p ： P 粘贴

按R可以替换字符

文本替换

设置行号

临时设置：

set nu 设置行号

set nonu 取消设置行号

永久设置：

vim /etc/vimrc 在这个文件后面追加 set nu

vim 打开多个文件

以上下形势，打开两个文档：

vim -o 文件路径 文件路径 例：vim -o /etc/passwd /etc/hosts

以左右形势，打开两个文档：

vim -O 文件路径 文件路径 例：vim -O /etc/passwd /etc/hosts

输入 ：qa 一次退出所有打开的文件

ctrl+ww 在两文档之间进行切换编辑。大写 O 左右分屏，小写的 o 上下分屏

比较文件：

ctrl+v 进入可视块模式

进入 Visual 模式的批量修改，方法如下：

1)、ctrl+v 进入列编辑模式

2)、向下或向上移动光标，把需要注释、编辑的行的开头选中起来

3)、然后按大写的 I

4)、再插入注释符或者你需要插入的符号,比如”#”

5)、再按 Esc,就会全部注释或添加了
